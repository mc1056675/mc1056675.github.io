Meet Cubroid

Introducing Cubroid, the world’s easiest coding block set that allows children to explore the world of technology and gain exposure to coding! Through dynamic connective blocks and simple coding, Cubroid offers a fun and educational experience for children to express their creativity.

{ Features }

Wireless

A set of coding blocks without any visible lines or connecting pins.

Lego Compatible

Compatible with Legos for creative purposes, so children can innovate without restrictions.

Smartphone Coding

children to control and prompt movement in their robot creations.

Physical Computing

Equipped with actuators and sensor blocks that work without external connections.

Scratch

Cubroid can be connected to a block coding software program called "Scratch" from MIT Media Lab.

Usb Charging

All blocks of cubroid can be charged with USB. You can use for 5 hours or 10 hours in one charge.

{ The easiest coding block in the world }

Wireless based modular robotics

Introducing Cubroid, the world’s easiest coding block set that allows children to explore the world of technology and gain exposure to coding! Through dynamic connective blocks and simple coding, Cubroid offers a fun and educational experience for children to express their creativity.

{ Coding App }

We currently provide our coding and control application on Androids and iPhones. In order to program the movements of your robot, use the simple coding function.

{ Control App }

Our app also allows you to directly control your robot through our virtual remote control.

{ Scratch Mode }

Cubroid can be connected to a block coding software program called "Scratch" from MIT Media Lab. Scratch is an in-depth coding application that will prompt you to take your coding skills to the next level. Cubroid can be operated through the Scratch coding platform when the master block is connected to a PC with a USB cable. Scratch Mode is recommended for more advanced users that are working to really hone in on their coding abilities.

{ Lego Compatibility }

Connect with Legos for endless creativity!

Cubroid blocks are seamlessly compatible with Lego blocks! In order to diminish the amount of restrictions that young creators face, we wanted them to be able to connect with Legos to enhance the structures of their own robots.

{ Master Block }

Control Blocks between sensor and actuator block

The Master Block is the control center of virtually any robot that one can build with Cubroid! With the Master Block, you can easily switch between the different modes and functions.
